# 🌸 Coming Soon Page – Chabi's Portfolio

A simple, elegant **“Coming Soon” landing page** designed to build anticipation for an upcoming portfolio launch. This page includes a countdown timer, a notification button, and a soft, friendly aesthetic.

---

## Features

- Live countdown timer (Days, Hours, Minutes, Seconds)
- Soft, aesthetic design
- "Notify Me" button for user interaction
- Responsive layout (mobile-friendly)
- Friendly and welcoming messaging

---

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/charbileigh/pink-themed-coming-soon-page.git

```

Or download the ZIP file and extract it.

---

### 2. Open the Project

Open `index.html` in your browser:

```bash
open index.html
```

Or simply double-click the file.

---

## Customisation

### You Can Update The Content Below For Your Personal Project After Cloning

Modify the content in `index.html`:

```html
<h1>Coming Soon</h1>
<p>Something exciting is on the way...</p>
```

---

### Styling

Update `styles.css` to customise:

- Colors
- Fonts
- Layout
- Animations

### Notify Button Functionality

The **Notify Me** button can be extended to:

- Collect email addresses
- Connect to a backend service
- Integrate with tools like Mailchimp or Firebase
